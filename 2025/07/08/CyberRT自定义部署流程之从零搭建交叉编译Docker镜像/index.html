    • /workspace实际是宿主机的目录；

    • docker commit不会保存这些改动 ❌。

📌 举例说明

  • 有挂载： 你在容器内/workspace修改的所有内容，其实是修改的宿主机/home/lxp/code，不属于容器，docker commit不会打包它。
  • 无挂载（纯容器内）： 创建或修改/workspace的文件，commit后会保存在镜像里。
